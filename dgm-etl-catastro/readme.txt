# DGM ETL Catastro

Procesos ETL para la gestión del Catastro Minero.

## Componentes

- etl → scripts de extracción y transformación
- pipelines → ejecución de procesos
- config → configuración de rutas y parámetros
- logs → registros de ejecución
- tests → pruebas

## Uso

Ejemplo:

##python etl/ETL-EXCEL-CMBASE.py
